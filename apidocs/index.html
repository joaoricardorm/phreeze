Phreeze API Docs

Instructions for generating Phreeze API documentation.

If you would rather not generate the API docs yourself then you can view the pre-generated API docs online at http://phreeze.com/docs/

API documentation for Phreeze can be generated using PhpDocumentor. The documentation is not included with the Phreeze distribution however you can generate it yourself. In order to generate API docs, you must have PhpDocumentor installed. Instructions for installing PhpDocumentor are available on the PhpDocumentor home page.

A shell script is located in this directory named generate.sh. This is a convenience script that will clean out the output directory and generate the documentation based on the comments in the source code files. The default setup expects you to have PhpDocumentor version 1. If you are using version 2 then you will need to edit generate.sh to comment out the v1 code and uncomment the v2 code.

Once PhpDocumentor is installed you can generate the docs with the following command:

cd /path/to/phreeze/apidocs
./generate.sh

Once the documentation has been generated it will be available in the html subdirectory of this folder.
